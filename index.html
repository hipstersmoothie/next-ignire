next-ignite

The most flexible documentation website you've ever used.

Check out the docs

ignite is a thin wrapper around next.js and next-mdx-enhanced. It is essentially two things:

  • A CLI for using ignite as a documentation generator
  • A convention and a set of functions. The ignite package exports everything you need to use it in any next.js application!
