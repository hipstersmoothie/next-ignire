Customizing the Sidebar

While it's cool you don't have to configure a Sidebar, you'll probably quickly find that you want some order. Luckily ignite makes this super easy. All you have to do is create a file _sidebar.md inside pages/docs and start organizing you docs.

This can be as simple as a list of links:

- [Getting Started](./getting-started)
