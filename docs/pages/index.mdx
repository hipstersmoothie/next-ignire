`ignite` is a thin wrapper around next-js and `next-mdx-enhanced`.
It is esentially two things:

- A CLI for using `ignite` as a documentation generator 
- A convention and a set of functions. The `ignite` package exports everthing you need to use it in any `next.js` application!
